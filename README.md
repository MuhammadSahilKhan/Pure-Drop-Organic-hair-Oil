# ICT Project Website

## 📌 Project Overview
This project is a simple and responsive website developed as part of our 1st Semester ICT course. It is built using HTML, CSS, and JavaScript to demonstrate the fundamentals of web development. The website features a clean design, easy navigation, and interactive elements.

## 🚀 Features
- Responsive web design
- Multiple web pages
- User-friendly interface
- Simple navigation
- Interactive elements using JavaScript

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
project/
│── index.html
│── about.html
│── contact.html
│── css/
│   └── style.css
│── js/
│   └── script.js
│── images/
└── README.md
```

## ▶️ How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your web browser.

## 📚 Learning Objectives
- Build a multi-page website
- Apply HTML, CSS, and JavaScript concepts
- Practice responsive web design
- Improve front-end development skills
